## Installation

Install with npm

```bash
  npm install @iconsans/vue
```

Install with pnpm

```bash
  pnpm add @iconsans/vue
```

Install with yarn

```bash
  yarn add @iconsans/vue
```

## Usage/Examples

```javascript
<script setup lang="ts">
    import { Play as PlayBold } from "@iconsans/vue/bold";
    import { Play as PlayLinear } from "@iconsans/vue/linear";
</script>


<template>
    <PlayBold />
    <PlayLinear />
</template>
```
